## 二、策略规则

### 入场 (Sniper Entry)

```
唯一入场信号: oi_accumulation
  条件: OI 4h变动 >15% 且 OI↑ 但 价格↓
  含义: 主力在吸筹，散户在抛售 → 做多信号

辅助确认 (不单独触发入场):
  - lsr_reversal (+20): 多空比反转 加分
  - lsr_bullish (+10): 多头增强 加分
  - lsr_extreme_bearish (+15): 空头拥挤 加分

禁止入场:
  - oi_short_squeeze: 信号滞后，回测9%胜率
  - near_support 单信号: 28%胜率，需强确认
  - vol_oi_high: 门控条件，不计分
  - taker_trending_up: 43%胜率，噪声

最低门槛:
  - FinalScore > 15 (复合分足够高)
  - OI > $5M (流动性足够)
  - 24h涨幅 < 10% (不追涨)
```

### 出场 (Sniper Exit)

```
止损 (SL): 入场价 - 1×ATR(4h)
  - ATR 是波动率自适应的 — 波动大的币止损宽，波动小的币止损紧
  - 止损设置在交易所端 (stop-market order)

止盈 (TP): 入场价 + 3×ATR(4h)
  - 3:1 风险回报比 (AiT 最低要求)
  - 止盈设置在交易所端 (take-profit order)

时间止损: 最大持仓 4h (1根 4h K 线)
  - 回测数据: 1h收益+0.251%, 4h收益-0.156%
  - 最优出场窗口: 1-2h
  - 超过4h强制平仓

信号失效止损:
  - 如果入场后4h OI转为下降 → 信号失效，平仓
  - 如果入场后LSR从多头翻空 → 趋势反转，平仓
```

### 仓位管理

```
最大同时持仓: 2 个
单笔仓位: 总资金 × 50% (分2笔)
杠杆: 3x (回测验证过)
扫描间隔: 4h (与K线周期一致)
宁缺勿滥: Top-10 中强信号标的 < 3 → 观望
```

### 预期指标 (基于回测)

```
信号频率: ~2笔/天 (oi_accumulation 在 90 笔/3天 = 30笔，约10笔/天，
         但仅 33/90 = 37% 有 oi_accumulation → ~3笔/天)
         加上 FinalScore>15 过滤 → 预期 ~1-2笔/天

胜率: 52% (oi_accumulation 回测)
均收: +2.095% × 3杠杆 = +6.29% (单笔)
3:1 R:R → 赢+18.87% 亏-6.29%
期望值: 0.52 × 18.87 + 0.48 × (-6.29) = +9.81 - 3.02 = +6.79%/笔
日均: +6.79% × 1.5笔 = +10.19%/天 (理论值，实际会打折)

风控:
  最大单亏: -6.29% (3杠杆 × 1×ATR止损)
  连亏3次: -18.87% → 暂停交易
  日亏>10%: 停止当日交易
```

## 三、与 AiT 引擎集成

### 工作流

```
每 4h (扫描间隔):
  1. Hunter GetHunterList() → Top-30 评分
  2. 过滤: 仅保留 oi_accumulation 标签的标的
  3. 二次过滤: FinalScore > 15, OI > $5M, 24h涨幅 < 10%
  4. 排序: FinalScore 降序
  5. 取 Top-2 → 发送给 AI 引擎
  6. AI 引擎: 检查 K 线/RSI/BOLL → 确认入场
  7. 执行: 开多 + 设 SL/TP (交易所端)
  8. 持仓监控: 每 1min 检查回撤 (auto_trader_risk)
  9. 出场: SL/TP 触发 或 4h 超时平仓
```

### AI Prompt 要点

```
角色: Hunter Sniper — 精准猎杀手
入场标准: 只做 oi_accumulation (主力吸筹) 信号
止损: 1×ATR(4h), 设置在交易所端
止盈: 3×ATR(4h), 设置在交易所端
持仓上限: 4小时
杠杆: 3x
风险回报比: ≥3:1
```

## 四、风险提示

1. **信号频率低**: 约 1-2 笔/天，不适合高频交易者
2. **连续亏损可能**: 52% 胜率意味着可能连续亏 3-4 笔
3. **ATR 止损不保证**: 极端行情可能滑穿止损价
4. **杠杆风险**: 3x 杠杆下 -1×ATR = -3% 账户回撤
5. **信号衰减**: 回测基于近 3 天数据，信号有效性可能随市场环境变化
