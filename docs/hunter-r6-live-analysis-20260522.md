### 3.3 ✅ OI Short Squeeze 检测

**FIDAUSDT**: OI 4h delta = **-10.89%**, 价格 +17.75% → 触发 `oi_short_squeeze` (+45分)

```
FIDAUSDT 评分分解:
  位置分: +0 (距支撑 4.2×ATR, 不在支撑位)
  OI 分: +45 (oi_short_squeeze: OI↓10.89% + 价格↑17.75%)
  聪明钱: +0 (LSR 中性 1.015→1.001)
  复合: clamp((0+45)/2 + 0×0.65, 0, 75) = 22.5
  最终: 22.5 × 1.0 × 1.0 = 22.5
```

**BEATUSDT**: OI 4h delta = **-5.35%**, 价格 +13.88% → 触发 `oi_squeeze_moderate` (+20分)

两个 squeeze 信号都被正确捕获。FIDA 排名从 R5 的 #3 升至 R6 的 #2（因为 #1 CLU 的 lsr_reversal + near_support 组合更强）。

### 3.4 ✅ ELP 极端亏损保护
